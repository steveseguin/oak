Oakridge Omni

A Meshtastic node in Mississauga. It has a website now.

About This Node

A LoRa radio strapped to a mast in south Mississauga, shouting into the void on 906.875 MHz. Part of the growing Meshtastic mesh in the GTA. It relays messages, it doesn't judge.

Hardware

  • Device: Heltec ESP32 V3
  • Antenna: 1.5m Omni
  • Mount: Mast-mounted

Location

  • City: Mississauga
  • Region: Greater Toronto Area
  • Province: Ontario, Canada

RF Equipment

  • Cable: LMR-600 N-Type Male to N-Type Male (6ft)
  • Antenna: Barracuda 915MHz 8dBi Omni-Directional
  • Frequency: 906.875 MHz (NA)

Technical Details

  • Protocol: LoRa Spread Spectrum
  • Range: Up to 20km line of sight
  • Power: USB-C powered with battery backup
  • Enclosure: LeMotech Junction Box, 4.7"x3.5"x2.7"

About the Technology

Meshtastic

An open-source communications platform that uses inexpensive LoRa radios to create long-range mesh networks. Perfect for hiking, emergency comms, or building community networks.

Learn More

ESP32 & LoRa

The Heltec ESP32 V3 combines WiFi, Bluetooth, and LoRa capabilities in one compact device. LoRa technology enables long-range, low-power wireless communication perfect for mesh networking.
